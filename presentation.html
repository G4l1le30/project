Demo: Alur Pengguna (Konsumen)

  • Melihat -> Rani membuka aplikasi & melihat UMKM terdekat/rekomendasi. (Data dari Firebase Realtime Database)
  • Mencari -> Rani mencari UMKM atau filter kategori.
  • Lokasi -> Rani beralih ke tampilan Peta. (Menggunakan Maps SDK for Android for Compose). Setiap UMKM direpresentasikan `Marker` dari Firebase.
  • Memilih -> Rani memilih UMKM & melihat halaman detail (menu, ulasan, lokasi). (Informasi dinamis dari Firebase)
  • Memesan -> Rani bisa memesan produk (jika makanan) atau melihat kontak.

Demo: Alur Pemilik UMKM

  • Login -> Ibu Siti login ke Owner Dashboard. (Otentikasi menggunakan Firebase Authentication).
  • Mengisi -> Mengisi profil bisnis: nama, deskripsi, foto, menu/layanan.
  • Menentukan -> Menentukan Lokasi: Ibu Siti mengetik alamat. (Sistem otomatis konversi alamat ke koordinat menggunakan `Geocoder` bawaan Android di *background thread* (Kotlin Coroutines). Ini metode aman & stabil.)
  • Menyimpan -> Data terupload ke Firebase Realtime Database. (Profil *live* di aplikasi berkat sifat *real-time* Firebase.)
